## Projects

**🤖 Automated Community Analytics Pipeline** · `n8n · Google Sheets · Dashboards`
Built, at the Hispanic Heritage Council of Western New York, an automated n8n workflow that triggers on new event registrations and survey entries, surfacing demographic insights in real time. Outputs directly inform program planning and outreach targeting for cultural education and senior care initiatives.

**🗞️ Agentic Fake News Detection** · `Apache Spark · FastAPI · MCP · Claude API`
End-to-end NLP system built on an Apache Spark MLlib ensemble (Logistic Regression, Gradient Boosting, Linear SVC) achieving 89.02% AUC. MCP integration with FastAPI exposes 6 AI-accessible tools, enabling real-time predictions and natural language querying directly through Claude Desktop.
